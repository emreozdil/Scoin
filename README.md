## Introduction
Sahibinden Coin iOS Application

Scoin App includes:
- Current Currency of Scoin
- Scoin History
- Scoin and Bitcoin Comparasion

## Requirements
- Swift 4
- Xcode 9.0+
- Charts
- Alamofire
- ObjectMapper
- AlamofireObjectMapper
